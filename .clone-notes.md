# Pastor Rhema — clone notes

Clone fiel da PV https://pastorrhema.com/ (nicho pastores/AI assistant, checkout Hotmart W103907822M).

## Estrutura
- `index.html` — página única, markup/CSS/copy idênticos ao original (inline `<style>`, sem framework).
- `assets/tl1.png` — screenshot do mockup de celular.
- `assets/dominus.js` — "Dominus Track Lite" (lib open-source de atribuição UTM: github.com/andersonfelipev/dominus-track). Decora automaticamente qualquer link pay.hotmart.com com utm/src/sck/xcod e auto-preenche forms.

## VSL / vídeo
- Embed Vturb (`vturb-smartplayer`) igual ao original: oid `acebec9c-d57e-4576-a08b-c8a8828588b2`, player `696a4f0b35c2a62b3c104a97`. Roda direto do CDN da converteai — **não precisa reupload**, funciona em qualquer domínio (testado em localhost).
- **Pitch/CTA na hora certa**: o vídeo tem `pitchTime: 234` (3:54) configurado no dashboard da Vturb (não no HTML) — isso vem junto de graça por reusar o mesmo ID do player, sem trabalho extra.
- Sem delay/reveal por JS — diferente do clone da Panaderia, aqui os CTAs e a PV inteira já aparecem visíveis desde o início (não tem `.esconder`/`displayHiddenElements`).
- **Gate de performance preservado**: um PerformanceObserver espera a request `facebook.com/tr` do pixel disparar antes de carregar os scripts pesados do player (fallback automático de 5s se o pixel não disparar). Assim que você trocar o pixel (ver abaixo), esse gate volta a funcionar exatamente como no original.

## Tracking — atualizado (2026-07-02)
